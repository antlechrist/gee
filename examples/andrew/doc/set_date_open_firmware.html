Set the date in Open Firmware

At the Open Firmware prompt, type:

decimal dev rtc <sec> <min> <hr> <day> <month> <year> set-time

So, for 20:46:00 Sunday, Feb 7th 2010:

decimal dev rtc 00 46 20 07 02 2010 set-time
